# 🚀 Quick Start - Run Your Dashboard NOW!

## Just 3 Commands!

### Terminal 1: Import Data & Start Backend

```bash
cd backend
node scripts\import_georgetown_data.js
npm start
```

### Terminal 2: Start Frontend

```bash
cd frontend
npm run dev
```

### Browser: Open Dashboard

```
http://localhost:3000/results
```
